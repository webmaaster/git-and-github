Most important Git Commands are:

  1. git desktop : To navigate to desktop
  2. git mkdir : Create a new folder
  3. git touch : Create a new file
  4. code . + enter: Open a code editor

Most important git commands to handle GitHub:

  1. git clone URL: To clone empty repository
  2. git add . + enter: Add all files & folders to stage area
  3. git commit -m "Message"
  4. git push: It will push all codes to GitHub

How to DEPLOY website on GitHub:

  1. Click on settings
  2. Scrol down until you see GitHub Pages
  3. Click on branch dropdown and select 'master'
  4. Click on save
  5. Copy the link and share with friends & family!
